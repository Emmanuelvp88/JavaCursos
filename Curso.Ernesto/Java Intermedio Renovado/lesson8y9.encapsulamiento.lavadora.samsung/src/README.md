Samsung solicita el desarrollo de una clase programada en java, para el funcionamiento logico de su nueva
linea de lavadoras, misma que puedan implementar sus programadores de manera sencilla en los programas
desarrollados dentro de samsung, con las suiguientes caracteristicas:

1.- Debe recibir los kilos de ropa y tipo de ropa atravez de argumentos.
2.- debe de realizar las hijas de llenado de agua, lavado y secado.
3.- La clase debe de estar correctamente encapsulado para evitar que los 
programadores de samsung utilicen metodos o variables que no sonnecesarios.
4.- El unico metodo disponible para importar, debe ser CicloFinalizado().
